# S.I.G.H.T. Component Changelogs

Because the S.I.G.H.T. ecosystem is composed of multiple discrete platforms that are built, released, and versioned on independent cadences, the monolithic project history has been split. 

Please view the dedicated Changelog for each component in its respective sub-directory:

- **[🖥️ Agent Desktop (`agent-desktop`)](/agent-desktop/CHANGELOG.md)**: Includes all history for the Rust/Tauri telemetry agent, SQLite auditing persistence, and local React UI overrides.
- **[🌐 Command Center Dashboard (`dashboard`)](/dashboard/CHANGELOG.md)**: Includes all history for the Next.js central management portal, Shadcn components, and UI/UX design changes.
- **[⚙️ Central API Server (`server`)](/server/CHANGELOG.md)**: Includes all history for the Go backend, PostgreSQL models, WebSocket hubs, and MQTT ingestion infrastructure.
